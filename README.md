# \# Smart Sericulture: Automated Disease Diagnosis in Silkworm using CNN

# 

# Smart Sericulture is a web-based system that uses Convolutional Neural Networks (CNN) to automatically diagnose diseases in silkworms and mulberry leaves, helping farmers detect infections early and reduce crop loss.

# 

# \## Features

# \- Silkworm disease detection (Healthy, Grasserie, Flacherie, Muscardine, Pebrine) from uploaded images

# \- Mulberry leaf disease detection (Leaf Rust, Leaf Spot, Stem Canker, Powdery Mildew, Root Knot)

# \- Causes, symptoms, and control measures for each detected disease, in English and Kannada

# \- Live cocoon market price scraper from the Karnataka Sericulture Department website

# \- Week-by-week silkworm rearing guide

# 

# \## Tech Stack

# \- Backend: Flask (Python)

# \- AI/ML: TensorFlow / Keras (CNN models)

# \- Image Processing: OpenCV

# \- Web Scraping: BeautifulSoup4

# \- Frontend: HTML, CSS, JavaScript

# 

# \## How It Works

# 1\. User uploads an image of a silkworm or mulberry leaf

# 2\. The image is preprocessed (resized, normalized)

# 3\. A trained CNN model classifies the image into a disease category

# 4\. The system returns the diagnosis along with causes, symptoms, and remedies
